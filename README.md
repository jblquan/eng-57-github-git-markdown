## Bash

Bash is a scripting language used by command line terminals.
We can use it to create file, navigate our computer or run programs.

### Basic Bash Commands

- ls (list) ls -a (hidden files)
- cd (go to)
- mkdir (create folder)
- touch (create file)
- rm (remove)
